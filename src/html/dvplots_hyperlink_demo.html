Direction and Velocity Plots Hyperlink demo

Each of the following links brings up an example of one of MATLAB's basic direction and velocity plots. The purpose of these examples is to show how the matlab: command can be used inside a published M-file to create a menu of MATLAB commands.

To use this file

  • Save the dvplots_hyperlink_demo.m file.
  • Open it and click the PUBLISH button or use the PUBLISH function from the command line
  • Open the resulting dvplots_hyperlink_demo.html file in the MATLAB web browser.
  • Try clicking on the various links

The plot types

Click one of the following links to bring up an example of that plot type.
